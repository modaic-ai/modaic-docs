title: Auto Classes
---

The `AutoProgram` class can be used to load an entire `PrecompiledProgram`, including Python modules and third-party dependencies, from Modaic Hub.

<Info>
If you are familiar with Hugging Face's `transformers` library, you will notice that `AutoProgram` is used similarly to the `AutoModel` class.
</Info>

## Load a Program

```python
from modaic import AutoProgram

program = AutoProgram.from_precompiled("entity-name/program-name")
```

## Load a Retriever

```python
from modaic import AutoRetriever

retriever = AutoRetriever.from_precompiled("entity-name/program-name")
```

## Load a Config

```python
from modaic import AutoConfig

config = AutoConfig.from_precompiled("entity-name/program-name")
```

## Load a Retriever and Program

```python
from modaic import AutoProgram, AutoRetriever

retriever = AutoRetriever.from_precompiled("entity-name/program-name")
program = AutoProgram.from_precompiled("entity-name/program-name", retriever=retriever)
```

## Override a Config
This will update the parameters of the config class that you specify.
```python
from modaic import AutoProgram

program = AutoProgram.from_precompiled("entity-name/program-name", config={"weather": "rainy"})
```

## Specify runtime parameters
You can specify runtime parameters for the program or retriever by passing them in as keyword arguments.
```python
from modaic import AutoRetriever
from retrievers import VectorDatabase, MilvusBackend

vector_database = VectorDatabase(MilvusBackend.from_local("index.db"))
retriever = AutoRetriever.from_precompiled("entity-name/program-name", vector_database=vector_database)
```

<Info>
This example uses the `mo-retrievers` SDK. Read more about it [here](https://docs.modaic.dev/retrievers).
</Info>

## Load a Specific Revision
You can load a specific revision of the program by passing in the `rev` argument. Rev can be a branch, tag, or commit hash (full or shortened).

```python
# branch
program = AutoProgram.from_precompiled("your-username/my-program", rev="dev")
# tag
program = AutoProgram.from_precompiled("your-username/my-program", rev="v1.0")
# commit hash
program = AutoProgram.from_precompiled("your-username/my-program", rev="918ad95")
```

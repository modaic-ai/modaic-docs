<Warning>`AutoProgram` is expirimental and subject to change. It is currently a work in progress, use at your own risk.</Warning>

The `AutoProgram` class, much like huggingface's `AutoModelForCausalLM` class, can be used to load an entire precompiled DSPy program from Modaic Hub. Unlike `AutoModelForCausalLM`, `AutoProgram` can load the entire program, including python modules and third party dependencies.

# Quick Start

```python
from modaic import AutoProgram

program = AutoProgram.from_precompiled("entity-name/program-name")
```

# Load a retriever

```python
from modaic import AutoProgram

retriever = AutoRetriever.from_precompiled("entity-name/program-name")
```

# Load a retriever and program

```python
from modaic import AutoProgram

retriever = AutoRetriever.from_precompiled("entity-name/program-name")
program = AutoProgram.from_precompiled("entity-name/program-name", retriever=retriever)
```

# Override a Config
This will update the parameters of the config class that you specify.
```python
from modaic import AutoProgram

program = AutoProgram.from_precompiled("entity-name/program-name", config={"weather": "rainy"})
```

# Specify runtime parameters
You can specify runtime parameters for the program by passing in a dictionary of parameters.
```python
from modaic import AutoProgram
from modaic.databases import VectorDatabase, MilvusBackend
from modaic.indexing import DummyEmbedder
from modaic.context.table import TableFile

vector_database = VectorDatabase(MilvusBackend.from_local("index.db"))
retriever = AutoRetriever.from_precompiled("entity-name/program-name", vector_database=vector_database)
program = AutoProgram.from_precompiled("entity-name/program-name", retriever=retriever, )
```
